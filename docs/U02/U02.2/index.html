1. ¿Qué es un String?

  • Un String es una secuencia de caracteres.
  • En Java, es un objeto de la clase String.

Ejemplos

String saludo = "Hola mundo";
String nombre = new String("Alejandro");

2. length()

  • length(): Devuelve la longitud del String.
String texto = "Java";
System.out.println(texto.length()); // 4

3. charAt(int index)

  • charAt(int index): Devuelve el carácter en la posición indicada (comenzando en 0).
String texto = "Java";
System.out.println(texto.charAt(1)); // 'a'

4. substring(int beginIndex) / substring(int beginIndex, int endIndex)

  • substring(int beginIndex): extrae desde una posición hasta el final.
  • substring(int beginIndex, int endIndex): extrae entre dos posiciones (begin inclusive, end exclusive).
String texto = "Programación";
System.out.println(texto.substring(6)); // "mación"
System.out.println(texto.substring(0, 6)); // "Progra"

5. toUpperCase() / toLowerCase()

  • toUpperCase(): convierte todos los caracteres a mayúsculas.
  • toLowerCase(): convierte todos los caracteres a minúsculas.
String texto1 = "java";
System.out.println(texto1.toUpperCase()); // "JAVA"

String texto2 = "JAVA";
System.out.println(texto2.toLowerCase()); // "java"

6. equals(String otro) / equalsIgnoreCase(String otro)

  • equals(String otro): compara contenido (sensible a mayúsculas).
  • equalsIgnoreCase(String otro): compara ignorando mayúsculas.
String a = "Hola";
String b = "hola";
System.out.println(a.equals(b)); // false
System.out.println(a.equalsIgnoreCase(b)); // true

7. contains(String subcadena)

  • contains(String subcadena): verifica si contiene una subcadena.
String texto = "Programación";
System.out.println(texto.contains("grama")); // true
System.out.println(texto.contains("Java")); // false

8. indexOf(String subcadena)

  • indexOf(String subcadena): devuelve la posición de la primera aparición de la subcadena (o -1 si no existe).
String texto = "Programación";
System.out.println(texto.indexOf("a")); // 5
System.out.println(texto.indexOf("z")); // -1

9. replace(char viejo, char nuevo) / replace(String viejo, String nuevo)

  • replace(): reemplaza caracteres o subcadenas.
String texto1 = "casa";
System.out.println(texto1.replace('a', 'o')); // "coso"

System.out.println("hola mundo".replace("mundo", "Java")); // "hola Java"

10. trim()

  • trim(): elimina espacios al inicio y al final.
String texto2 = "  hola  ";
System.out.println(texto2.trim()); // "hola"

11. compareTo(String otro) / compareToIgnoreCase(String otro)

  • compareTo(String otro): compara lexicográficamente; devuelve <0 si este < otro, 0 si iguales, >0 si este > otro.
  • compareToIgnoreCase(String otro): igual que compareTo pero ignorando mayúsculas/minúsculas.
String a = "abc";
String b = "bcd";
System.out.println(a.compareTo(b)); // negativo (por 'a' < 'b')
System.out.println(a.compareTo("abc")); // 0

String x = "Hola";
String y = "hola";
System.out.println(x.compareTo(y)); // distinto de 0 (sensible a mayúsculas)
System.out.println(x.compareToIgnoreCase(y)); // 0

12. Transformaciones entre tipos

Primitivo → String String → Primitivo
int: Integer.toString(n), String.valueOf(n), "" + n int: Integer.parseInt(s)
char: Character.toString(c), String.valueOf(c), "" + c char: s.charAt(0)
boolean: Boolean.toString(b), String.valueOf(b) boolean: Boolean.parseBoolean(s)
float: Float.toString(f), String.valueOf(f) float: Float.parseFloat(s) 
int n = 123;
// Transformar de int a String
String s1 = Integer.toString(n);     // "123"
String s2 = String.valueOf(n);       // "123"
String s3 = "" + n;                  // "123"

// Transformar de char a String
char c = "hola".charAt(0);            // 'h'
String cs = Character.toString(c);   // "h"

// Transformar de String a int
String numText = "456";
int m = Integer.parseInt(numText);   // 456
