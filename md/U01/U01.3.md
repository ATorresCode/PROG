## 1. Estructura básica de un programa
```java
/* Estructura de una clase de Java
 * Si no es clase principal, el método main no aparece
 */
public class NombreDeLaClase {
    // Declaración de los atributos de la clase
    // Declaración de los métodos de la clase

    // El método main indica dónde empieza la ejecución
    public static void main(String[] args) {
        // Declaración de las variables del método
        // Sentencias de ejecución del método
    }
}
```

### Comentarios y separadores
**Comentarios**
- `/* … */`  (bloque)
- `// …`     (línea)
- `/** … */` (documentación)

**Separadores**
- `()` listas de parámetros (definición y llamada a métodos)
- `{}` delimita bloques y valores iniciales de *arrays*
- `[]` declaración de *arrays* y acceso a sus elementos
- `;` separa instrucciones
- `,` separa identificadores/argumentos
- `.` separa elementos de un objeto (acceso a miembros)

## 2. Variables y tipos de datos
Una **variable** permite **almacenar datos**, resultados e intermedios. Toda variable tiene asociado un **tipo de datos** que:
- Define el **conjunto de valores** que puede almacenar.
- Determina las **operaciones** que pueden realizarse.

**Declaración en Java**
```java
Tipo Identificador;
```

### Tipos de datos
- **Primitivos**
  - numéricos: `byte`, `short`, `int`, `long`, `float`, `double`
  - carácter: `char`
  - lógico: `boolean`
- **Referencia**
  - `String`, *arrays*, clases e interfaces (se verán más adelante).

### Tipos de datos primitivos
**Numéricos**
- Enteros con diferentes tamaños (byte/short/int/long) y reales (`float`, `double`).

**Carácter**
| Nombre | Tamaño | Representa |
|---|---|---|
| `char` | 2 bytes | Caracteres (letras, números y símbolos). Java usa **Unicode** de 2 bytes. |

**Booleano**
| Nombre | Tamaño | Representa |
|---|---|---|
| `boolean` | 1 bit | Valores lógicos `true` / `false`. |

### Identificadores de variables I
Los **identificadores** son los nombres de variables, clases, interfaces, atributos y métodos.

**Reglas**
- Java **distingue mayúsculas y minúsculas**: `var1`, `Var1` y `VAR1` son distintos.
- Longitud prácticamente **ilimitada**.
- **No** puede ser palabra reservada ni `true`/`false`.
- **No** pueden duplicarse en el mismo **ámbito**.
- Debe **empezar por una letra**; pueden usarse letras latinas/hebreas/cirílicas, y también `$` (no recomendado) y `_` (subrayado).
- **No** usar espacios en blanco ni símbolos que coincidan con **operadores**.

### Identificadores de variables II (convenciones)
- Variables y métodos: **minúscula** inicial.
- Clases: **mayúscula** inicial.
- Si el identificador tiene varias palabras: primera en minúscula (salvo clases), resto con mayúscula inicial (**camelCase** / **UpperCamelCase**).
  - `variable`: `añoDeCreacion`
  - `clase`: `HolaMundo`

### Declaración de variables
```java
// Formas básicas
int edadPedro = 34;
float precioPatata = 1.2F, precioChoco = 2.3F;
char car = 'A', car2 = '\u0041'; // 0041 es el Unicode de 'A'

// Otras formas
int x;            // solo declaración
int y = 10;       // declaración + inicialización
int a, b, c;      // múltiples
int d = 1, e = 2; // múltiples con valores
```

### Inicialización de variables
Si una variable **no** se ha inicializado, tiene un **valor por defecto** (cuando es **miembro** de clase):
- Numéricos → `0`
- `char` → `'\u0000'`
- `boolean` → `false`
- Referenciales (objetos) → `null`

> Nota: Las **variables locales** deben inicializarse antes de usarse; no tienen valores por defecto.

### Asignación
La instrucción de **asignación** permite **dar** o **modificar** el valor de una variable.
```java
identificador = expresion;

int suma;         // declaración
char ch1, ch2 = 'u';
float f1 = 2.0F, f2; 

f2 = 34.67F;
suma = suma + 2;
```

### Ámbito de una variable
- El **ámbito** es la parte del programa donde una variable es **visible** y puede usarse.
- **Variables locales**: declaradas dentro del cuerpo de un método; visibles **solo** en ese método.
- Pueden declararse en cualquier punto del cuerpo, aunque es buena práctica **al principio**.
- También pueden declararse dentro de un **bloque** `{ … }`; serán visibles **solo** dentro del bloque.
- Las variables definidas en un bloque deben tener **nombres diferentes**.

### Constantes
Para declarar una constante se usa el modificador `final`.
```java
final double PI = 3.1415926536; // debe inicializarse al declararse
```

## 3. Literales
### Literales I
**Definición**: un *literal* es un **valor que se expresa a sí mismo**.

**Enteros**
- Decimal (base 10): `21`
- Octal (base 8): `025`
- Hexadecimal (base 16): `0x3A`
- Sufijo `L`/`l` para `long`: `21L`

**Reales**
- Notación decimal: `345.678`, `0.00056`
- Notación científica/exponencial: `3.45678e2`, `5.6e-4`
- Sufijos: `F`/`f` → `float`, `D`/`d` → `double`
  - Por defecto es `double`; para `float` usar sufijo `F`.

### Literales II — carácter
- Entre **comillas simples** `'…'`.
- Ejemplos: `'a'`, `'B'`, `'{'`, `'ñ'`, `'á'`.
- Código Unicode del carácter (octal/hexadecimal):
  - `'\141'` (octal) → `'a'`
  - `'\u0061'` (hex) → `'a'`
- **Secuencias de escape** más comunes:
  - `\'` comilla simple
  - `\"` comillas dobles
  - `\\` barra invertida
  - `\b` backspace
  - `\n` salto de línea
  - `\r` retorno de carro
  - `\t` tabulador

### Literales III — booleano y `String`
- Booleano: `true`, `false`.
- `String`: entre **comillas dobles** `"…"`.
```java
System.out.println("Primera línea\nSegunda línea del string\n");
System.out.println("Hol\u0061"); // "Hola"
```

## 4. Conversión de tipos
Al asignar: `identificador = expresion;` ambos deben ser del **mismo tipo** o **compatibles**.

**Conversión implícita (ampliación)**
```
short → int → long → float → double
```

**Conversión explícita (*casting*)**
```java
num = (int) 34.56;
```
También mediante **funciones** de librerías cuando corresponda.

## 5. Operadores
### Operadores unarios
- **Signo**: `+expresion`, `-expresion`
- **Incremento/Decremento**: `++`, `--` (pre o post)
```java
int valor, i = 5;
i++;     // i = 6 (equivale a i = i + 1)
--i;     // i = 5 (equivale a i = i - 1)

valor = i++;  // valor = 5, i = 6
valor = ++i;  // valor = 7, i = 7
```

### Operadores aritméticos I
```java
int x;
x = 52 * 17;
x = 120 / 4;
x = 67 + 34;
x = 32 - 12;
x = 7 % 2;   // resto de división entera
```

### Operadores aritméticos II (lista)
- `+` suma
- `-` resta
- `*` multiplicación
- `/` división
- `%` resto (módulo)

### Operadores de comparación
- Símbolos que evalúan expresiones condicionales y devuelven **boolean**.
- Sintaxis: `exp1 operador exp2`.

**Relación de operadores**
- `<`  menor que
- `<=` menor o igual que
- `>`  mayor que
- `>=` mayor o igual que
- `==` igual a
- `!=` distinto de

**Ejemplos**
```java
boolean testResult;
testResult = (45 < 35);
testResult = (45 == 45);
testResult = (4 != 3);
testResult = ('a' > 'b');

int cantidad;
boolean pedidoGrande;
pedidoGrande = cantidad > 1000;
```

### Operadores lógicos I–II
- Combinan expresiones y devuelven **boolean**.
- Sintaxis: `exp1 operador exp2`.

**Lista**
- `&&` (AND): ambas expresiones deben ser `true` para resultar `true`.
- `||` (OR): si **una** es `true`, el resultado es `true`.
- `!`  (NOT): negación lógica.

**Ejemplo**
```java
edad > 18 && sexo == 'H'
```

### Operadores lógicos III — Tablas de verdad
**AND**

| Expres1 | Expres2 | Resultado |
|---|---|---|
| true | true  | true  |
| true | false | false |
| false| true  | false |
| false| false | false |

**OR**

| Expres1 | Expres2 | Resultado |
|---|---|---|
| true | true  | true  |
| true | false | true  |
| false| true  | true  |
| false| false | false |

**NOT**

| Expresión | Resultado |
|---|---|
| true  | false |
| false | true  |

### Combinar comparación y lógica
Podemos combinar operadores de comparación y lógicos con instrucciones condicionales.
```java
edad <= 25 && edad >= 14
```

### Operadores de asignación
```java
edad = 34;
edad += 1;  // edad = edad + 1
edad -= 3;  // edad = edad - 3
edad *= 2;  // edad = edad * 2
edad /= 2;  // edad = edad / 2
edad %= 2;  // edad = edad % 2
```

### Operador de concatenación
Genera una **cadena** a partir de otras dos:
```java
"Hola" + ", " + "buenos días"
```

### Prioridad de operadores I
Cuando hay varias operaciones en una expresión, se evalúa con un **orden predeterminado**:
1. **Paréntesis** (de dentro a fuera)
2. **Unarios**
3. **Aritméticos**
   - Multiplicativos: `*`, `/`, `%`
   - Sumativos: `+`, `-`
4. **Comparativos/Relacionales**
5. **Lógicos**
   - `!` (NOT)
   - `&&` (AND)
   - `||` (OR)
6. **Asignación**

> Cuando aparecen operadores de la **misma prioridad**, el compilador evalúa **de izquierda a derecha**.

### Prioridad de operadores II — Ejemplo
```java
a = -3 + 5 + 2 * 4 - 6 / 4 * 3 - 5 % 2;
// Orden de evaluación (referencia de la diapositiva): 1 6 7 2 8 3 4 9 5
// Valor final de a = 6
```

## 6. Entrada / Salida
### Salida de datos por pantalla
```java
System.out.print("Se imprime este mensaje sin el retorno de carro");
System.out.println("Se imprime este mensaje con un retorno de carro");
```

### Entrada de datos del teclado I
```java
char c = (char) System.in.read(); // lee un solo carácter
```
Esta forma es **poco práctica** para números o cadenas.

### Entrada de datos del teclado II — `Scanner`
```java
// Importamos el paquete donde está la clase
import java.util.Scanner;

public class Exemple {
    public static void main(String[] args) {
        int primerNum;
        // Declaramos el objeto lector de la clase Scanner
        Scanner lector = new Scanner(System.in);
        System.out.print("Escriu un número i polsa retorn: ");
        // Se lee un valor entero y se espera al retorno
        primerNum = lector.nextInt();
        lector.nextLine(); // limpia el salto de línea
    }
}
```

### Entrada de datos del teclado III — métodos de `Scanner`
| Método              | Tipo leído |
|---------------------|------------|
| `nextByte()`        | `byte`     |
| `nextShort()`       | `short`    |
| `nextInt()`         | `int`      |
| `nextLong()`        | `long`     |
| `nextFloat()`       | `float`    |
| `nextDouble()`      | `double`   |
| `nextBoolean()`     | `boolean`  |
| `next()`            | `String`   |
| `nextLine()`        | `String`   |

### Ejemplo completo — *Adivinanza*
```java
import java.util.*;
class Adivinanza {
    public static void main(String[] args) {
        Scanner tcl = new Scanner(System.in);
        int valor;
        System.out.println("Piensa un numero");
        tcl.nextLine();
        System.out.println("Multiplicalo por 5");
        tcl.nextLine();
        System.out.println("Sumale 6");
        tcl.nextLine();
        System.out.println("Multiplıcalo por 4");
        tcl.nextLine();
        System.out.println("Sumale 9");
        tcl.nextLine();
        System.out.println("Multiplıcalo por 5");
        tcl.nextLine();
        System.out.println("Escribe el resultado");
        valor = tcl.nextInt();
        System.out.println("El numero que habıas pensado es: ");
        System.out.println((valor - 165) / 100);
    }
}
```

## 7. Tipos Enumerados
Son **conjuntos de valores constantes** para los que no existe un tipo predefinido (por ejemplo: días de la semana, estaciones, meses).
```java
public class Semana {
    public enum DiaSemana {LUNES, MARTES, MIERCOLES, JUEVES, VIERNES, SABADO, DOMINGO}
    public static void main(String[] args) {
        DiaSemana hoy = DiaSemana.JUEVES;
        DiaSemana ultimo = DiaSemana.DOMINGO;
        System.out.println("Hoy es " + hoy + "\nY el ultimo dia es " + ultimo);
    }
}
```

## 8. Para ampliar
### Clase `Math`
**Constantes**
- `Math.E = 2.7182818284590452354`
- `Math.PI = 3.14159265358979323846`

**Funciones de redondeo** (con `x` de tipo `double`)
- `ceil(x)`: entero más pequeño **≥** `x`
- `floor(x)`: entero más grande **≤** `x`
- `round(x)`: redondea `x` al entero más próximo

**Funciones trigonométricas** (argumentos en **radianes**)
- `sin(x)` — seno
- `cos(x)` — coseno
- `asin(x)` — arco seno (`x` entre `-1` y `1`)
- `acos(x)` — arco coseno (`x` entre `-1` y `1`)
- `atan(x)` — arco tangente

**Otras funciones**
- `abs(x)` — valor absoluto
- `exp(x)` — *e* elevado a `x`
- `log(x)` — logaritmo natural (`x` ≥ 0)
- `max(x,y)` — máximo
- `min(x,y)` — mínimo
- `pow(x,y)` — `x` elevado a `y`
- `random()` — número pseudoaleatorio entre `0.0` y `1.0`
- `sqrt(x)` — raíz cuadrada (`x` ≥ 0)

### Tipo `String` — uso sencillo
```java
String frase, palabra, linea;
frase = "En un lugar de la Mancha de cuyo nombre…";
// … solicitamos al usuario una frase
// … y la frase de entrada es: Oh! es terrible
palabra = tcl.next();        // palabra = "Oh!"
linea   = tcl.nextLine();    // linea   = "Oh! es terrible"
```

### Sistema de tipos
Dos categorías generales:
- **Tipo simple o valor**: la variable **contiene directamente** sus datos; cada variable tiene **su propia copia**.
- **Tipo referencia**: la variable contiene una **referencia/puntero** al objeto; dos variables pueden referirse al **mismo objeto**.

Los **tipos simples** (primitivos) tienen **clases envoltorio** equivalentes:
- `byte`    → `java.lang.Byte`
- `short`   → `java.lang.Short`
- `int`     → `java.lang.Integer`
- `long`    → `java.lang.Long`
- `float`   → `java.lang.Float`
- `double`  → `java.lang.Double`
- `char`    → `java.lang.Character`
- `boolean` → `java.lang.Boolean`

> Nota: existen además tipos comunes, cadenas, matrices (*arrays*), clases y enumeraciones definidas por el usuario.
