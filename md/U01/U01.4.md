# Uso de estructuras de decisión o condicionales

---

## Expresiones condicionales

- Incluyen una condición que se evalúa como `true` o `false`
- Incluyen un operador para especificar el resultado de la condición

```java
if (ContraseñaEntrada == ContraseñaGuardada) {
    // La condición es true
} else {
    // La condición es false
}
```

---

## Estructuras de decisión

- Instrucción `if`
- Instrucción `if...else`
- Instrucción `if...else if...`
- Instrucción `switch...case`
- Operador ternario `? :`

---

## Instrucción `if`

```java
if (condición)
    sentencia;

// o bien

if (condición) {
    sentencia1;
    sentencia2;
    // ...
    sentenciaN;
}
```

---

## Uso de `if`

- Se utiliza para una decisión `true` o `false`
- Si la condición es `true`, se ejecutan las instrucciones siguientes
- Si la condición es `false`, no se ejecutan

```java
if (carácter == 'a') {
    System.out.println("Se ha leído el carácter a");
}
```

[Más información](https://www.w3schools.com/java/java_conditions.asp)

---

## Instrucción `if...else`

```java
if (condición)
    sentencia1;
else
    sentencia2;

// o bien

if (condición) {
    sentencia11;
    // ...
    sentencia1N;
} else {
    sentencia21;
    // ...
    sentencia2N;
}
```

---

## Uso de `if...else`

- Decisión con dos opciones como mínimo
- Si la condición es `true`, se ejecutan las instrucciones tras `if`
- Si la condición es `false`, se ejecutan las instrucciones tras `else`

```java
if (edad < 18) {
    System.out.println("No puedes salir a la hora del patio");
} else {
    System.out.println("Puedes salir");
}
```

---

## Detalles de `if...else`

- La parte `else` es opcional
- El bloque requiere `{}` si contiene más de una instrucción
- Un bloque puede ser la sentencia vacía `;` (no se ejecuta nada)

---

## Instrucción `if...else if...`

- Se utiliza para anidar instrucciones de decisión

```java
if (mes == 1)
    print("enero");
else if (mes == 2)
    print("febrero");
else if (mes == 3)
    print("marzo");
else
    print("a partir de abril");
```

---

## Instrucción `switch...case`

- Sintaxis compacta para seleccionar un bloque según un valor
- Alternativa a `if...else` anidados
- Funciona sobre enteros, booleanos o caracteres
- Las condiciones deben ser excluyentes

---

## Ejemplo de `switch...case`

```java
switch (mes) {
   case 1:
       print("enero");
       break;
   case 2:
       print("febrero");
       break;
   // ...
   default:
       print("no sé");
}
```

---

## Detalles de `switch...case`

- Las sentencias `break` terminan la ejecución del bloque
- Si no hay `break`, el código siguiente se ejecuta
- El `default` es opcional

[Más información](https://www.w3schools.com/java/java_switch.asp)

---

## Ejemplo de `switch...case` con varios casos

```java
switch (mes) {
   case 1: case 3: case 5: case 7:
   case 8: case 10: case 12:
       dias = 31;
       break;
   case 4: case 6: case 9: case 11:
       dias = 30;
       break;
   case 2:
       if (bisiesto)
          dias = 29;
       else
          dias = 28;
       break;
   default:
       dias = 0;
}
```

---

## Directrices para elegir una estructura de decisión

- `if`: controlar la ejecución de un único bloque de código
- `if...else`: controlar dos secciones mutuamente excluyentes
- `if...else if...`: controlar múltiples condiciones excluyentes de forma secuencial
- `switch...case`: lista de valores posibles

---

## Operador ternario `? :`

- Forma compacta de decidir entre dos valores

```java
variable = condición ? valor1 : valor2;
```

Equivalente a:

```java
if (condición)
    variable = valor1;
else
    variable = valor2;
```

# Estructuras de repetición o bucles

---

## Estructuras de bucle condicionales

- Instrucción `while`
- Instrucción `do...while`
- Instrucción `for`
- Instrucciones `break` y `continue`

---

## Instrucción `while`

```java
while (condición)
    sentencia;

// o bien

while (condición) {
    sentencia1;
    sentencia2;
    // ...
}
```

---

## Uso de `while`

- Ejecuta el código solo si la condición es `true`
- Se repite hasta que la condición sea `false`
- La condición se evalúa al principio

```java
int fact = 1, n = 10;
while (n > 0)
    fact *= n--;
```

[Más información](https://www.w3schools.com/java/java_while_loop.asp)

---

## Instrucción `do...while`

```java
do {
    sentencia;
} while (condición);

// o bien

do {
    sentencia1;
    sentencia2;
    // ...
} while (condición);
```

---

## Uso de `do...while`

- Ejecuta el código y luego evalúa la condición
- Se ejecuta al menos una vez
- La condición se evalúa al final

```java
double nota, suma = 0.0, media, n = 0;
do {
    nota = lector.nextDouble();
    suma += nota;
    n++;
} while (nota < 5.0);
media = suma / n;
```

---

## Instrucción `for`

```java
for (inicialización; condición; actualización)
    acción;

// o bien

for (inicialización; condición; actualización) {
    acción1;
    acción2;
    // ...
}
```

---

## Uso de `for`

- Se utiliza cuando se conoce el número de repeticiones

```java
int fact = 1;
for (n = m; n > 0; n--)
    fact *= n;

for (i = 0; i < 10; i++)
    System.out.print(i);
```

[Más información](https://www.w3schools.com/java/java_for_loop.asp)

---

## Detalles de `for`

- El contador es una variable local al bucle
- Los componentes sintácticos (inicialización, condición y actualización) pueden omitirse si no son necesarios, pero los puntos y comas deben mantenerse. Por ejemplo:

```java
for ( ; ; ) {
   // bucle infinito
}
```

```java
for (int i = 0; i < 10; i++)
    System.out.print(i);
```

---

## `break` y `continue`

- `break`: termina el bucle
- `continue`: comienza una nueva repetición, omitiendo el resto del bloque actual

> Se recomienda no usar `break` y `continue` salvo que sea evidente su necesidad

[Más información](https://www.w3schools.com/java/java_break.asp)

---

## Ejemplo de `break` y `continue`

```java
for (int i = 0; i < 10; i++) {
   if (i == 5)
      break; // Sale del bucle cuando i es 5
   if (i % 2 == 0)
      continue; // Salta a la siguiente iteración si i es par
   System.out.println(i); // Solo imprime valores impares menores que 5
}
```

---

## Directrices para elegir una estructura de repetición

- `while`: cuando no se conoce el número de iteraciones y se desea repetir mientras se cumpla una condición.
- `do...while`: similar a `while`, pero garantiza al menos una iteración.
- `for`: cuando se conoce el número exacto de iteraciones.
- `foreach`: para iterar sobre elementos de una colección o array (lo veremos más adelante).
