## 1. La clase `Math`

Para utilizar las funciones y constantes de la clase `Math`, no es necesario importar ninguna librería adicional, ya que forma parte del paquete `java.lang` que se importa automáticamente.

**Ejemplo de uso:**

```java
double raiz = Math.sqrt(25);      // 5.0
double angulo = Math.toRadians(30);
double seno = Math.sin(angulo);   // 0.5
```

---

## 1. La clase `Math`

**Constantes:**

- `Math.E` = 2.7182818284590452354
- `Math.PI` = 3.14159265358979323846

**Funciones de redondeo (`double x`):**

- `Math.ceil(x)` → entero más pequeño ≥ x
- `Math.floor(x)` → entero más grande ≤ x
- `Math.round(x)` → entero más próximo

## 1. La clase `Math`

**Funciones trigonométricas:**

- `Math.sin(x)` → seno (radianes)
- `Math.cos(x)` → coseno (radianes)
- `Math.asin(x)` → arco seno (x entre -1 y 1)
- `Math.acos(x)` → arco coseno (x entre -1 y 1)
- `Math.atan(x)` → arco tangente

## 1. La clase `Math`

**Otras funciones matemáticas:**

- `Math.abs(x)` → valor absoluto
- `Math.exp(x)` → e elevado a x
- `Math.log(x)` → logaritmo natural
- `Math.max(x, y)` → mayor entre x e y

- `Math.pow(x, y)` → x elevado a y

## 2. El tipo de datos `String`

**Métodos de lectura `next()` y `nextLine()`**

```java
String palabra, linea;

// Entrada del usuario: "Oh! es terrible"
palabra = tcl.next();     // palabra = "Oh!"
linea = tcl.nextLine();   // linea = "Oh! es terrible"
```

---

## 2. El tipo de datos `String`

**Métodos útiles de la clase `String`:**

- `length()` → devuelve la longitud de la cadena
- `charAt(i)` → carácter en la posición `i`
- `substring(a, b)` → subcadena entre las posiciones `a` y `b`
- `toUpperCase()` / `toLowerCase()` → mayúsculas/minúsculas
- `equals(str)` → compara cadenas

---

## 2. El tipo de datos `String`

**Ejemplos de uso:**

```java
String saludo = "Hola Mundo";

System.out.println(saludo.length()); // 10

System.out.println(saludo.charAt(1)); // 'o'

System.out.println(saludo.substring(0, 4)); // "Hola"

System.out.println(saludo.toUpperCase()); // "HOLA MUNDO"
System.out.println(saludo.toLowerCase()); // "hola mundo"

System.out.println(saludo.equals("Hola Mundo")); // true
System.out.println(saludo.equals("hola mundo")); // false
```

## 3. Tratamiento de fechas y horas con `java.time`

La librería `java.time` (disponible desde Java 8) proporciona clases para manejar fechas y horas de manera sencilla y eficiente.

- Para trabajar con fechas:

```java
import java.time.LocalDate;

// Fecha actual
LocalDate hoy = LocalDate.now();
System.out.println(hoy);

// Fecha específica (por ejemplo, 15 de mayo de 2000)
LocalDate fechaEspecifica = LocalDate.of(2000, 5, 15);
System.out.println(fechaEspecifica); // Salida: 2000-05-15
```

## 3. Tratamiento de fechas y horas con `java.time`

- Para calcular la diferencia entre dos fechas usando `Period`:

```java
import java.time.LocalDate;
import java.time.Period;

LocalDate fechaNacimiento = LocalDate.of(2000, 5, 15);
LocalDate hoy = LocalDate.now();

Period periodo = Period.between(fechaNacimiento, hoy);
System.out.printf("Han pasado %d años, %d meses y %d días\n",
  periodo.getYears(), periodo.getMonths(), periodo.getDays());
```

## 4. Salida de datos y formato

- Para mostrar información por pantalla se utilizan principalmente tres métodos:
  - `System.out.println()`: imprime el texto y añade un salto de línea al final.
  - `System.out.print()`: imprime el texto sin añadir salto de línea.
  - `System.out.printf()`: permite imprimir texto formateado.

---

## 4. Salida de datos y formato

**Uso de `printf()`:**

Permite especificar el formato de los datos a mostrar, usando especificadores como:

- `%d` para enteros
- `%f` para números decimales (puedes indicar el número de decimales, por ejemplo `%.2f`)
- `%s` para cadenas de texto (Strings)
- `%n` para salto de línea (como `\n`)

---

## 4. Salida de datos y formato

**Ejemplo:**

```java
int edad = 20;
double nota = 8.75;
String nombre = "Ana";

System.out.printf("Nombre: %s, Edad: %d, Nota: %.1f%n", nombre, edad, nota);
// Salida: Nombre: Ana, Edad: 20, Nota: 8.8
```

## 5. Argumentos en la línea de comandos

- Se reciben como un array (aún no los hemos visto) de Strings en el método `main`
- `args[0]` es el primer argumento, `args[1]` el segundo, etc.

```java
public class Ejemplo {
  public static void main(String[] args) {
    System.out.println("Primer argumento: " + args[0]);
  }
}
```
---

## 5. Argumentos en la línea de comandos
- A la hora de compilar y ejecutar el programa desde la consola, se pasan los argumentos separados por espacios.:

```sh
javac Ejemplo.java
java Ejemplo hola
```

La salida será:

```
Primer argumento: hola
```
