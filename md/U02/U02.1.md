## 1. ¿Qué es un método?

- Conjunto de instrucciones referenciadas por un identificador  
- Puede ser llamado desde distintos puntos del programa  
- Existen dos tipos de métodos:
  - **Funciones**: devuelven un valor
  - **Procedimientos**: no devuelven un valor


### Ejemplos de métodos en Java

```java
double rx = Math.sqrt(78);
int i = entrada.nextInt();
System.out.println("Hola a todos");
```

- **Identificador**: `sqrt`, `nextInt`, `println`  
- **Parámetros** entre paréntesis  
- Algunos devuelven **resultado** (`sqrt` y `nextInt`), otros no (`println`)

## 3. Invocación de métodos

### Llamada a métodos

- Un método puede llamar (o invocar) a otro, y se transfiere el control hasta que termina  
- Un método se puede llamar a sí mismo (**recursividad**)  
- Como sabemos, en Java, el programa siempre empieza a ejecutarse en el método `main`, que podrá llamar a otros métodos

---

### Diagrama de llamada

- M1 ➡️ M2: M1 llama a M2  
  - Se pueden transferir datos (**parámetros**)  
- M1 ⬅️ M2: M2 devuelve el control a M1  
  - Puede devolver un dato de salida

## 4. Crear un método

### Componentes de un método

```java
static tipo_devuelto nombre([argumentos]) {
  // instrucciones
  return valor;
}
```

- `tipo_devuelto`: tipo de dato del valor que devuelve (`void` si no devuelve nada)
- `argumentos`: tipo de dato y nombre de cada parámetro dentro del método (opcional)
- `return`: especifica el valor devuelto (inexistente si `void`)



### Ejemplo

- Método `suma` que devuelve la suma de dos enteros

```java
static int suma(int a, int b) {
  return a + b;
}
```

## 5. Valor de retorno

### Ejemplo

```java
static int maximo(int x, int y) {
  if (x >= y) return x;
  else return y;
}
```

- `return` especifica el valor devuelto  
- Devuelve el control al método invocador

## 6. Llamar a un método

### Ejemplo de M1 que llama a M2

- **Argumentos actuales**: contiene los valores con los que M1 invoca a M2 `M2(arg1, arg2, ..., argN);`
- **Argumentos formales**: especificados en la cabecera de M2 `M2(tipo arg1, tipo arg2, ..., tipo argN)`

- Los argumentos actuales deben coincidir en número y tipo con los formales


### Ejemplo 1

- Implementación del método `suma`
```java
static int suma(int a, int b) {
  return a + b;
}
```
- Distintas llamadas al método `suma` desde `main`
```java
public static void main(String[] args) {
  int y = suma(5, 3); // y vale 8
  int z = suma(7, 10); // z vale 17
}
```


### Ejemplo 2

- Implementación del método `maximo`
```java
static int maximo(int x, int y) {
  if (x >= y) return x;
  else return y;
}
```


### Ejemplo 2

- Obtención del máximo de cuatro números
```java
public static void main(String[] args) {
  int a = 5, b = 12, c = 8, d = 3;
  int max1 = maximo(a, b); // max1 vale 12
  int max2 = maximo(c, d); // max2 vale 8
  int max = maximo(max1, max2); // max vale 12
}
```


### Ejemplo 2

- Otra manera de obtener el máximo de los cuatro números
```java
public static void main(String[] args) {
  int a = 5, b = 12, c = 8, d = 3;
  int max = maximo(maximo(a, b), maximo(c, d));
}
```

---

### Cabecera vs llamada

| Cabecera (args. formales) | Llamada (args. actuales) |
|---------|---------|
| `int suma(int a, int b)` | `suma(2, 4)` |
| `int maximo(int x, int y)` | `maximo(7, 9)` |
| `void imprime(int a, float b, char c)` | `imprime(numero, 3.14, 'x')` |

## 7. Paso de argumentos

### Tipos de paso

- **Por valor**: El método no modifica la variable original  
  - En Java, los **tipos primitivos** (byte, short, int, long, float, double, boolean, char, e incluso String) se pasan por valor
- **Por referencia**: El método puede modificar la variable original
  - En Java, los **objetos** se pasan por referencia. Lo veremos más adelante.


### Ejemplo paso por valor

```java
int a = 3;
System.out.println("antes: a=" + a);  // antes: a=3
funcion(a);
System.out.println("después: a=" + a);  // después: a=3

public static void funcion(int x) {
  x = 5;
  System.out.println("dentro: a=" + x); // dentro: a=5
}
```
- La variable `a` no se modifica


### Comparativa paso por valor vs referencia

| Mecanismo | Explicación | Implicaciones | Ventaja |
|----------|-------------|---------------|---------|
| Por valor | Se pasa una copia | No se modifica el original | Protege la variable |
| Por referencia | Se pasa la dirección | Se puede modificar | Permite devolver valores como argumentos |

## 8. Métodos sobrecargados

### Ejemplo

- Mismo nombre, diferente número/tipo de argumentos  

```java
int suma(int a, int b) {
  return a + b;
}

int suma(int a, int b, int c) {
  return a + b + c;
}
```

---

### Ejemplo de uso

- Podemos llamar al método que queramos según los argumentos que le pasemos

```java
public static void main(String[] args) {
  int x = suma(5, 10);  // llama al primer método y vale 15
  int y = suma(5, 10, 15);  // llama al segundo método y vale 30
}
```

## 9. Ámbito de definición de métodos

- Parámetros y variables locales son accesibles solo dentro del método  
- Una clase puede definir y usar sus propios métodos  
- No hay restricciones en el orden de los métodos. En el caso de existir método `main`, puede estar en cualquier posición

```java
public class Ejemplo {
  static void metodo1() { ... }
  static void metodo2() { ... }
  public static void main(String[] args) { ... }
}
```

## 10. Uso de métodos de otras clases

### Métodos estáticos

- Podemos usar métodos de otra clase, siempre que sean `static` (cuando veamos la POO, veremos cómo usar métodos no estáticos de otras clases)
- Para usar un método de otra clase, se escribe `NombreClase.nombreMetodo(...)`
- Ejemplos de métodos `static` que ya hemos usado: `Math.sqrt()`, `System.out.println()`


### Implementación

```java
public class Matematicas {
  public static int factorial(int n) {
    int resultado = 1;
    for (int i = 2; i <= n; i++) resultado *= i;
    return resultado;
  }

  public static boolean esPrimo(int n) {
    if (n <= 1) return false;
    for (int i = 2; i <= Math.sqrt(n); i++) if (n % i == 0) return false;
    return true;
  }
}
```

### Llamada a métodos estáticos

```java

public class PruebaMatematicas {
  public static void main(String[] args) {
    int numero = 5;
    System.out.println("Factorial de " + numero + ": " + Matematicas.factorial(numero));
    System.out.println(numero + " es primo: " + Matematicas.esPrimo(numero));
  }
}
```
